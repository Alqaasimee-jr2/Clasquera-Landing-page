# Clasquera Landing Page

Lesson Accountability System for private schools. This is a high-fidelity, responsive landing page built with **Next.js 16**, **TypeScript**, and **Tailwind CSS v4**.

## Project Description
Clasquera records each lesson, transcribes it, and scores how closely it followed the lesson plan—so you see teacher performance without leaving your office.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features
- **WhatsApp Integration**: Pilot program requests redirected to WhatsApp.
- **Localized Data**: Realistic Nigerian school performance metrics.
- **Premium Design**: Modern typography, glassmorphism, and smooth animations.
- **Fully Responsive**: Optimized for both desktop and mobile devices.
